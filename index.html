What is Protparser?

Protparser is a tool designed for finding MRI protocols in a quick and easy fashion.  You can either download our script to run it in a customizable fashion, or check out our standard protocols page for quick use. To download the code with instructions for installation and use click the 'Store' button above. For our standard protocols page click the  'Updates' link above.
